# Fundación Adulam de Bolivia (sitio estático)

## Cómo usar
1) Abrí `index.html` en tu navegador (doble click).
2) Editá textos y datos donde veas `EDITAR` o `__________`.
3) Para el QR:
   - reemplazá `assets/img/qr-placeholder.svg` por tu archivo QR (por ejemplo `qr.png`)
   - y actualizá el `src` en `index.html` y `donaciones.html`.

## Publicar
- Funciona en cualquier hosting estático (Netlify, Vercel, GitHub Pages, hosting común).
- No requiere PHP ni base de datos.

## Archivos principales
- index.html
- quienes-somos.html
- programas.html
